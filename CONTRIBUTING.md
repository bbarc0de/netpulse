# Contributing to NetPulse

By submitting a contribution, you represent that you have the right to submit it and agree that it is licensed under AGPL-3.0-only with the rest of the project. Preserve copyright, license, privacy, measurement-methodology, and non-affiliation notices. Do not submit credentials, personal data, generated build output, or copied material without compatible rights.

Contributions that display a network metric must identify the real source, formula, sample count/duration where applicable, and browser limitations. Do not add random, simulated, fallback, or placeholder values presented as measurements.

Before opening a pull request, run `npm ci` and `npm run check`. Security-sensitive changes should include misuse tests and updates to the threat model. Report vulnerabilities privately under [SECURITY.md](SECURITY.md), not in a public issue.

Contribution terms and governance should receive qualified legal review before the project relies on them for contested ownership questions.
